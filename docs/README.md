## Quick Start

```
IdleClicker → Generate Assets   (creates sprites)
IdleClicker → Setup Scene       (builds MainScene.unity)
File → Build Settings → Android → Switch Platform
File → Build Settings → Build
```

## Project Layout

```
Assets/
  Scripts/         GameAssembly.asmdef — runtime scripts
  Editor/          Editor-only tools (SceneBuilder, LintChecker, AssetGenerator)
  Tests/EditMode/  Tests.EditMode.asmdef — NUnit edit-mode tests
  Resources/       Sprites/, Audio/ (optional), bloodidle_secrets.txt (gitignored)
  Scenes/          MainScene.unity (generated by SceneBuilder)
```
